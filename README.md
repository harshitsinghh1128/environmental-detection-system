# AI Environmental Detection System

An AI-powered IoT environmental monitoring system built using:

- ESP32
- FastAPI
- Next.js
- Machine Learning
- OpenWeather API
- Real-time Analytics

## Features

- Real-time temperature monitoring
- Humidity analysis
- Rain prediction
- Pollution monitoring
- Historical analytics dashboard
- IoT-ready architecture

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Recharts
- Lucide Icons

### Backend
- FastAPI
- Python
- SQLite
- Scikit-learn

### Hardware
- ESP32
- DHT11/DHT22
- MQ135
- Rain Sensor
